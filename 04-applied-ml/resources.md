# Resources: Applied Machine Learning & Model Evaluation

## Documentation

- [scikit-learn User Guide](https://scikit-learn.org/stable/user_guide.html) — comprehensive guide to all ML algorithms
- [scikit-learn API Reference](https://scikit-learn.org/stable/modules/classes.html) — quick reference for every class and function
- [scikit-learn Metrics](https://scikit-learn.org/stable/modules/model_evaluation.html) — complete evaluation metrics documentation
- [NumPy Documentation](https://numpy.org/doc/) — array operations and numerical computing
- [Pandas Documentation](https://pandas.pydata.org/docs/) — data manipulation and analysis
- [Matplotlib Documentation](https://matplotlib.org/stable/contents.html) — plotting and visualization

## Tutorials and Courses

- [Kaggle Learn: Intro to Machine Learning](https://www.kaggle.com/learn/intro-to-machine-learning) — hands-on beginner course
- [Kaggle Learn: Intermediate Machine Learning](https://www.kaggle.com/learn/intermediate-machine-learning) — feature engineering and advanced models
- [Kaggle Learn: Model Validation](https://www.kaggle.com/learn/model-validation) — cross-validation and train/test splits
- [Google's Machine Learning Crash Course](https://developers.google.com/machine-learning/crash-course) — Google's free ML course
- [Fast.ai Practical ML](https://course.fast.ai/) — top-down practical approach to ML

## Datasets for Practice

- [Kaggle Datasets](https://www.kaggle.com/datasets) — thousands of datasets for practice
- [UCI Machine Learning Repository](https://archive.ics.uci.edu/ml/index.php) — classic ML datasets
- [OpenML](https://www.openml.org/) — open dataset repository with sklearn integration
- [Google Dataset Search](https://datasetsearch.research.google.com/) — search engine for datasets

## Model Evaluation Guides

- [Precision-Recall Trade-off (Google ML Crash Course)](https://developers.google.com/machine-learning/crash-course/classification/precision-and-recall)
- [Confusion Matrix Explained (Scikit-learn)](https://scikit-learn.org/stable/auto_examples/model_selection/plot_confusion_matrix.html)
- [ROC Curve Tutorial](https://scikit-learn.org/stable/modules/model_evaluation.html#roc-metrics)
- [Cross-Validation Best Practices](https://scikit-learn.org/stable/modules/cross_validation.html)

## Books

- *Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow* by Aurelien Geron — practical ML guide
- *An Introduction to Statistical Learning (ISLR)* by James, Witten, Hastie, Tibshirani — free textbook, excellent theory
- *The Elements of Statistical Learning* by Hastie, Tibshirani, Friedman — advanced statistical learning (free PDF)
